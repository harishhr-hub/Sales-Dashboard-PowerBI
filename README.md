# 📊 Sales Dashboard using Power BI

An interactive Sales Dashboard developed using **Microsoft Power BI** to analyze sales performance, customer behavior, product trends, and regional sales. The dashboard provides dynamic visualizations and business insights using KPI cards, DAX measures, Power Query, slicers, and drill-through reports.

---

## 📌 Project Overview

This dashboard helps businesses monitor key sales metrics and make data-driven decisions by providing an interactive and visually appealing report. Users can filter data using slicers, explore customer details through drill-through functionality, and analyze sales across different categories, regions, and time periods.

---

## 🚀 Features

- 📈 KPI Cards
  - Total Sales
  - Total Orders
  - Total Customers
  - Total Categories

- 📊 Interactive Visualizations
  - Sales by Category
  - Monthly Sales Trend
  - Sales by Region
  - Sales by Segment
  - Top States by Sales
  - Top 5 Customers
  - Top Products

- 🎛 Interactive Filters (Slicers)
  - Region
  - Category
  - Segment
  - Order Date

- 🔍 Drill-through Report
  - Customer Details page for detailed customer purchase analysis

- ⚡ Dynamic filtering and cross-highlighting across all visuals

## ▶️ How to Use

1. Download the repository.
2. Open **SalesDashboard.pbix** using Microsoft Power BI Desktop.
3. Refresh the data if required.
4. Explore the dashboard using slicers.
5. Right-click a customer in the **Top 5 Customers** chart and select **Drill-through → Customer Details**.
6. Use the Back button to return to the main dashboard.
