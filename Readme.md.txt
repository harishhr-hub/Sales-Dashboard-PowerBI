# 📊 Sales Dashboard using Power BI

## 📌 Overview
This project is an interactive Sales Dashboard developed using Microsoft Power BI to analyze sales performance and provide actionable business insights. The dashboard enables users to monitor key performance indicators (KPIs), explore regional sales trends, analyze customer and product performance, and interact with data through slicers and drill-through functionality.

## 🚀 Features
- 📈 KPI Cards (Total Sales, Orders, Customers, Categories)
- 📊 Sales by Category
- 📉 Monthly Sales Trend
- 🌍 Sales by Region
- 🏢 Sales by Segment
- ⭐ Top Products Analysis
- 👥 Top 5 Customers
- 🗺️ Top States by Sales
- 🔍 Interactive Slicers (Region, Category, Segment, Order Date)
- 🔄 Drill-through Customer Details Page
- ⚡ Dynamic filtering and cross-highlighting

## 🛠️ Technologies Used
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Microsoft Excel

## 📂 Dataset
Global Superstore Sales Dataset

## 📈 Business Insights
- Identified top-performing products and customers.
- Compared sales performance across regions and segments.
- Analyzed monthly sales trends.
- Enabled detailed customer-level analysis using drill-through.
- Built an interactive dashboard for faster business decision-making.
